## Simple E-Commerce Store

A modern e-commerce web application built with React and Vite. It features user authentication, product browsing, cart management, checkout, and order confirmation.

### Features
- User registration and login
- Product listing and detail pages
- Add to cart and cart management
- Checkout and order confirmation
- Toast notifications for user feedback
- Responsive design

### Technologies Used
- React
- Vite
- TypeScript
- Context API

### Setup & Installation
1. Navigate to the `Simple E-Commerce store` folder:
   ```cmd
   cd "Simple E-Commerce store"
   ```
2. Install dependencies:
   ```cmd
   npm install
   ```
3. Start the development server:
   ```cmd
   npm run dev
   ```

## Project Management Tool

A Kanban-style project management tool for organizing tasks across boards and columns. Includes task creation, editing, and drag-and-drop functionality.

### Features
- Create, edit, and delete tasks
- Organize tasks into columns (To Do, In Progress, Done)
- Drag-and-drop task movement
- Modal dialogs for task details
- Responsive UI

### Technologies Used
- React
- Vite
- TypeScript

### Setup & Installation
1. Navigate to the `Project Management Tool` folder:
   ```cmd
   cd "Project Management Tool"
   ```
2. Install dependencies:
   ```cmd
   npm install
   ```
3. Start the development server:
   ```cmd
   npm run dev
   ```
